feat: Refactor InteractiveApp to Scene-Based Architecture

This commit completes Phase 2 of the InteractiveApp refactoring.

Key changes include:
- Replaced the monolithic `AppMode` enum and state machine with a scene-based architecture.
- Extracted all major application modes into discrete `Scene` classes.
- Introduced a `SceneId` enum for type-safe scene transitions.
- Resolved circular import dependencies between scene files.
- Updated existing tests to be compatible with the new architecture.
- Removed obsolete test files whose logic is now covered by or irrelevant to the new scene-specific tests.
- Restored token visibility toggle functionality within the new `ViewingScene` and `MapScene`.
